# Lab5-6
Лабораторная работа 5-6

ФИО: Яблоков Николай Дмитриевич
Группа: СКБ-251

***Описание***
Консольное приложение для управления личной библиотекой с использованием в разработке объектно-ориентированного программирования. Приложение позволяет добавлять книги и пользователей в библиотеку, осуществлять возврат и выдачу книг, сохранять состояние библиотеки в файл.

***Требования:***
C++17 и выше
CMake 3.10 и выше

Сборка:

***MacOS/Linux:***

Создание папки для сборки

```
mkdir build
cd build
```
Генерация Makefile

`cmake ..`

Компиляция 

`make`

Запуск

`./library`

***Windows:***

```
mkdir build
cd build
cmake -G "MinGW Makefiles" ..
mingw32-make
.\library.exe
```


***Реализованные функции***

Управление книгами:
-добавление
-поиск по isbn
-просмотр каталога

Управление пользователями:
-добавление
-поиск по имени
-просмотр всех пользователей

Операции:
-выдача книг
-возврат книг

Сохранение состояние:
-загрузка данных при запуске
-сохранение при выходе из программы
-ручное сохранение
-форматированное сохранение в текстовом файле

Обработка ошибок:
-Множество проверок при взаимодействии пользовталя с книгами, всевозможные проверки на способность пользователя взять/отдать книгу (аналогично с книгами)
